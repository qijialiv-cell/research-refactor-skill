# Research Code Refactoring Skill

**[English](README_EN.md)** | **[中文](README_CN.md)**

---

A structured code refactoring framework for academic research projects (R / Stata / Python), designed for use with **Claude Code**.

> **Quick Install:**
> ```bash
> mkdir -p ~/.claude/skills/research-refactor
> cp SKILL.md ~/.claude/skills/research-refactor/
> cp progress.md ~/.claude/skills/research-refactor/
> ```
